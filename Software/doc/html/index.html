DC motor control shield V1.0 Documentation

DC motor control shield V1.0 is Nucleo-64 shield which can control position/velocity/torque of brushed DC motor FA-130RA.

Key functions

Most users only need to refer or change the following functions (these are related to motor control performance or output data).

  • static void MajorControlLoop(void)
    Major control loop function (Period : 200[us])
    Users can change control mode (Position/Velociry/Torque), command or gains.
    Also, users can change current control gains.

  • static void MinorControlLoop(void)
    Minor control loop function (Period : 50[us])

  • static void PositionControl(float PosCmd, float VelCmd, float P_Gain, float I_Gain, float D_Gain)

  • static void VelocityControl(float VelCmd, float P_Gain, float I_Gain)

  • static void TorqueControl(float Command)

  • static void configCurrentControl(bool isEnabled, float P_Gain, float I_Gain)

  • void SerialCommunicationTask(void const *argument)
    Low priority task that communicates with UART
    Users can send parameters such as command, response, and gains to the PC via ST-Link on Nucleo.
    User can display or save these parameters in real time by using a terminal (e.g. Tera Term) or a serial plotter (e.g. Arduino IDE, CPLT).
